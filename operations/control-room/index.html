실시간 관제

차량 위치, 탑승 현황, 교통 정보를 관제 센터에서 실시간 모니터링. 돌발 상황에 즉각 대응합니다.

관제 시스템이 포함하는 것

GPS 기반 차량 위치 추적

모든 운행 차량의 위치를 지도 위에서 실시간으로 확인합니다.

탑승 현황 실시간 집계

정류장별, 노선별 탑승 인원을 실시간 집계하여 수급을 조절합니다.

돌발 상황 자동 알림

지연, 사고, 노쇼 등 이슈 발생 시 관제 센터와 현장에 자동 알림합니다.

현장 즉시 대응

관제 센터 ↔ 현장 매니저 ↔ 기사 간 실시간 소통으로 즉각 대응합니다.
